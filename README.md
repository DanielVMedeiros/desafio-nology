# Desafio NOLOGY
## 🎯 Objetivo
### Criar um sistema para calcular o cashback para uma fintech

## 💻 Tecnologias utilizadas

- **Backend:** Python - FastAPI

- **Frontend:** JavaScript - React

- **Banco de Dados:** MySQL

## Link de Acesso
https://desafio-nology-rouge.vercel.app/

## Pergunta 1 
O código para resolver a primeira pergunta é o calculaCashback, um código único python que retorna o cálculo do cashback 

## Pergunta 5 
O Resto do repositório foi feito para resolver essa pergunta, fiz com FastAPI(Python) no Back-end e o Front-end fiz com react

## Como rodar?
### Front-end
```bash
Primeiro Comando - cd frontend
```
```bash
Segundo Comando - npm i
```
```bash
Terceiro Comando - npm start 
```

### Back-end 
Primeiro Comando -
```bash
cd backend 
```
Segundo Comando - 
```bash
pip install -r requirements.txt
```
Terceiro Comando - 
```bash
fastapi dev 
```

## Comentários 
É necessário um .env na raiz do projeto para realizar testes locais ou apontar para o local na hora das requisições.
